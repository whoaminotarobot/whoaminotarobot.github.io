Verdant Node

Project Overview

Verdant Node is an advanced plant stress monitoring system developed as part of the Verdant Spectrum team project. It combines multiple environmental sensors to provide real-time insights into plant health and stress conditions.

Key Features

  • Multi-sensor environmental monitoring (temperature, humidity, light)
  • Real-time stress detection algorithms
  • Data logging and analysis capabilities
  • Low-power operation for extended monitoring
  • Arduino-based architecture for accessibility

Technologies Used

  • Arduino Uno/Nano microcontroller
  • DHT11 temperature/humidity sensor
  • LDR (Light Dependent Resistor) for light intensity
  • Soil moisture sensors
  • SD card module for data logging
  • WiFi module for remote monitoring (optional)

My Contribution

As a core team member of Verdant Spectrum, I architected the sensor integration framework, developed calibration algorithms, and implemented real-time data processing logic. I also created the data visualization dashboard and troubleshot hardware integration challenges.

Status & Impact

Verdant Node is actively being deployed in agricultural and research settings. The system has successfully demonstrated 95% accuracy in early stress detection, enabling farmers to intervene before significant crop damage occurs.

Biosonification System

Project Overview

The Biosonification System converts plant bioelectric signals into audible frequencies, creating a revolutionary approach to plant health monitoring. By "listening" to plants, we can detect stress, disease, and environmental changes in real-time through audio feedback.

How It Works

  • Bioelectric signal acquisition from plant tissues using specialized electrodes
  • Signal amplification and filtering through analog conditioning circuits
  • ADC conversion and digital signal processing on Arduino
  • Real-time frequency mapping and sonification algorithms in Python
  • Audio output through speakers for intuitive feedback

Technologies & Components

  • Arduino Uno for signal acquisition and processing
  • Op-amp circuits for bioelectric signal amplification
  • Python for advanced signal processing and sonification
  • FFT (Fast Fourier Transform) for frequency analysis
  • PyAudio for real-time audio generation
  • Custom electrode design for bioelectric coupling

My Contribution

I designed the signal acquisition circuit, developed the Arduino firmware for ADC and preprocessing, created the Python signal processing pipeline, and implemented the sonification algorithm that maps bioelectric patterns to musical frequencies. I also conducted extensive testing to validate the system's accuracy and responsiveness.

Innovation & Impact

This project won first place at PEC Hacks 2025 for its innovative approach to human-plant interaction. It demonstrates how IoT and signal processing can create new sensory modalities for environmental monitoring. Applications extend to agriculture, horticulture, research, and education.

Current Status

The system is in active development with ongoing refinements to signal processing algorithms, electrode design optimization, and exploration of machine learning for pattern recognition in plant health states.

Bee Hive Monitoring System

Project Overview

An intelligent IoT system designed to monitor bee hive health indicators in real-time. Using multiple environmental and acoustic sensors, the system detects anomalies that indicate disease, pest infection, or queen absence.

Monitoring Parameters

  • Temperature and humidity inside the hive
  • Acoustic analysis of bee activity patterns
  • Hive entrance activity tracking
  • Weight monitoring for resource assessment
  • Vibration patterns of bee colonies

Technologies Used

  • Arduino Nano for compact deployment
  • DHT22 temperature/humidity sensors
  • MEMS microphones for acoustic monitoring
  • Vibration sensors for colony activity
  • ESP8266 WiFi module for cloud connectivity
  • MQTT protocol for data transmission

My Role

As supporting engineer, I contributed to sensor integration, developed the acoustic signal analysis algorithms, and helped implement the anomaly detection system. I also assisted in field testing and data validation.

Real-World Impact

This system helps beekeepers make informed decisions about hive health, reducing losses due to preventable diseases and improving colony management practices. It demonstrates IoT applications in agricultural technology and environmental conservation.

Safety Monitoring System

Project Overview

A comprehensive safety monitoring system that uses multi-sensor arrays to detect potential hazards through vibration and acoustic analysis. The system can identify equipment failures, structural issues, and anomalous activities in real-time.

Detection Capabilities

  • Vibration anomaly detection for mechanical failures
  • Acoustic event classification for impact detection
  • Pattern recognition for behavioral anomalies
  • Real-time alert generation with prioritization
  • Historical data logging for trend analysis

System Architecture

  • Arduino-based sensor hub with multiple input channels
  • ADXL345 accelerometer for vibration measurement
  • Analog microphone with amplification circuit
  • DSP algorithms for signal feature extraction
  • SD card logging for long-term data retention

My Contribution

I developed the core signal processing algorithms for both vibration and acoustic analysis, implemented the anomaly detection classifier, designed the alert system with priority levels, and created the data logging infrastructure. This was the implementation backbone of the project.

Deployment & Results

The system is currently deployed and demonstrates excellent performance in identifying equipment anomalies with minimal false positives. It successfully predicted several potential failures before they became critical, validating the approach.
